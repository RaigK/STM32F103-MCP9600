# STM32F103-MCP9600
OLED Thermocouple Demo
The board with an STM32F103CBT and an SH1106 OLED (128x64) comes from a defective soldering station. I used this material to test the MCP9600 thermocouple ADC.The display shows two temperatures as well as the date and time of the real-time clock. A menu for settings can be operated using a rotary knob.
